# Ockam

Ockam is a collection of protocols and toolkits for building connected
systems that you can trust.

This folder contains the Elixir implementation of Ockam.

## Build

1. Lint

```
../../gradlew lint
```

2. Build

```
../../gradlew build
```

3. Test

```
../../gradlew test
```

4. Clean

```
../../gradlew clean
```
